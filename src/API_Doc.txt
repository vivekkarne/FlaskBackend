Readme for APIs

POST - add a new review record, cannot update via POST, need to use put for updating a record 

PUT - to insert new reviews or update if the review already exists

GET - Gets all review for a product index_

DELETE - Deletes a review for the given product_id , user_id pair

GET /prod - Gets all the fields for a given product_id

GET /user - Gets all the fields for a given user_id

Reference: https://www.elastic.co/guide/en/elasticsearch/reference/current/docs-index_.html
